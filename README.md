# mac_dotfiles

My custom dotfiles for macOS.

## Install

```shell
$ bash install.sh
```

## Features

### Custom mod
- VIM highlight
- SSH connection reuse
- Styling shell prompt (Solarized Dark theme)
- Handy aliases

### Automatic software installation
- Install homebrew
- Install applications (List in `Brewfile`)

### Custom setting
- Change some application's language
